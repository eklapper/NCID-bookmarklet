HuffPost NCID Tracking Code Appender

Use these bookmarklets to append an Omniture NCID tracking code to HuffPost Greece articles that are to be posted from the 24Media social accounts.

Simply drag each bookmarklet to your bookmark bar, and click that bookmarklet when visiting a story you would like to append the tracking code to. The URL is then ready to be shared on Facebook or Twitter.
